How to Play
  1. Place: Tap Place to create a Tier 1 building on an empty tile.
  2. Select: Tap a building to select it, then tap an empty tile to move it.
  3. Merge: Tap a building then tap another of the same tier to merge → tier +1.
  4. Income: Buildings generate coins per second. Higher tier = more coins.
  5. Upgrades: Buy boosts (income, discounts, bigger grid).
  6. Offline: Earn coins while away (if enabled).
  7. Prestige: Reset for a permanent multiplier.
Pro tip: keep merging one “chain” up to high tiers for big income spikes.
